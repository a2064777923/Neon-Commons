    Create lib/fighting/manager.js following the RacingRoomManager pattern from lib/racing/manager.js. Adapt for fighting game specifics:

    **1. Singleton export:**
    ```javascript
    function getFightingRoomManager() {
      if (!global.fightingRoomManager) {
        global.fightingRoomManager = new FightingRoomManager();
      }
      return global.fightingRoomManager;
    }
    ```

    **2. Constructor:**
    - this.rooms = new Map()
    - this.io = null
    - this.reconnectGraceMs = 15000
    - this.reconnectTimers = new Map()
    - this.roomExpiryTimers = new Map()

    **3. Room lifecycle methods (adapt from RacingRoomManager):**
    - attachIo(io)
    - listRooms() / listPublicRooms()
    - countOpenRoomsByOwner(ownerId)
    - createRoom(owner, overrides = {}) — creates room with:
      - gameKey: "fighting"
      - fightPhase: "waiting"
      - characters: []
      - inputs: new Map()
      - roundWins: new Map()
      - currentRound: 1
      - arena: getDefaultArena()
      - config: { visibility, maxPlayers: 2, roundCount: overrides.roundCount || 3 }
    - getRoom(roomNo)
    - joinRoom(roomNo, user) — max 2 players per room
    - setReady(roomNo, userId, ready)
    - registerSocket(roomNo, userId, socket)
    - unregisterSocket(socketId)

    **4. Game start flow:**
    - maybeStartFight(room) — check all players ready + min 2 players
    - startCountdown(room) — 3-second countdown (same pattern as Racing)
    - startFight(room):
      - Set fightPhase = "fighting"
      - Create characters: `createCharacter(0, "right")`, `createCharacter(1, "left")` (facing each other)
      - Position at arena spawnPoints
      - Initialize roundWins Map: {0: 0, 1: 0}
      - Start 60Hz game loop: `room.loopTimer = setInterval(() => this.gameTick(room), 16.67)`

    **5. 60Hz game loop — gameTick(room):**
    - Guard: if fightPhase !== "fighting" return
    - room.tick += 1
    - Step 1: Process inputs → updateCharacterState for each player's character
    - Step 2: Apply physics → applyGravity + resolvePlatformCollision for each character
    - Step 3: Check hitbox/hurtbox collisions between all character pairs
      - checkHitboxCollision(char[i], char[j]) → applyDamage if hit and not invulnerable
      - checkHitboxCollision(char[j], char[i]) → applyDamage if hit and not invulnerable
      - Reset hitThisAttack at start of each tick (before collision checks)
    - Step 4: Check ring-out → checkRingOut for each character → handleRoundEnd
    - Step 5: Check KO → health <= 0 → set state to "ko" → handleRoundEnd
    - Step 6: Compute delta and broadcast:
      ```javascript
      const delta = computeDelta(room, room.previousState);
      room.previousState = delta;
      this.io?.to(getSocketRoom(room.roomNo)).emit(SOCKET_EVENTS.fighting.update, delta);
      ```

    **6. Round management:**
    - handleRoundEnd(room, winnerSeatIndex):
      - Stop game loop (clearInterval)
      - Increment roundWins for winner
      - Set fightPhase = "round_end"
      - Check match win: if roundWins[winner] > Math.floor(roundCount / 2):
        - Set fightPhase = "match_end"
        - Call finishMatch(room, winner)
      - Else: start next round after 3-second delay
        - Set fightPhase = "next_round"
        - Increment currentRound
        - Reset characters (health, position, state) but keep roundWins
        - After 3s delay: startFight again (new round)

    - finishMatch(room, winner):
      - Set room.state = "waiting", fightPhase = "waiting"
      - Build lastResult with winner info
      - Apply settlements (same pattern as Racing)
      - Reset players ready state
      - Clear characters, inputs, etc.
      - Emit room to all players

    **7. Input handling:**
    - submitInput(roomNo, userId, input):
      - Validate user is in room
      - Validate fightPhase === "fighting"
      - Validate and clamp input: { left, right, up, attack, heavy, block, dodge } — all booleans
      - Store in room.inputs.set(seatIndex, validatedInput)

    **8. Serialization:**
    - serializeRoom(room, viewerUserId) — same pattern as Racing:
      - Include room metadata, players, viewer, availability
      - Include match state (characters with pos, state, health, energy) when fighting
    - serializeRoomSummary(room) — for room list
    - emitRoom(room) — emit to all player sockets

    **9. Reconnection:**
    - registerSocket: mark seat connected, join socket room, emit current state
    - unregisterSocket: mark reconnecting, start grace timer
    - On reconnect: character state is preserved in room.characters; client receives full state on emitRoom

    **10. Helper:**
    - getSocketRoom(roomNo) returns `fighting:${roomNo}`
    - pushFeed(room, message, type) — same as Racing
    - createHumanSeat(user, seatIndex) — same as Racing
