  <action>Create `pages/pickred/[roomNo].js` — the Pick Red game room page. Follow the existing board room page pattern but adapted for Pick Red:

1. **Page structure:**
   - `getServerSideProps` — fetch room detail from `API_ROUTES.pickredRooms.detail(roomNo)`
   - `useRouter` for roomNo
   - `useEffect` for socket connection and state updates

2. **Socket connection:**
   - Connect to backend Socket.IO
   - Emit `pickred:subscribe` with roomNo and userId
   - Listen for `pickred:update` to receive serialized room state
   - Listen for `pickred:error` for error messages

3. **Game state display:**
   - `room.round.stage` — "waiting" | "playing" | "finished"
   - `room.round.tableCards` — face-up table cards
   - `room.round.lastDiscard` — last discarded card
   - `room.round.deckCount` — remaining cards in deck
   - `room.round.scores` — both players' scores
   - `room.round.currentTurn` — whose turn (0 or 1)
   - `room.round.turnEndsAt` — countdown timer target
   - `room.hands` — viewer's hand cards (from serializeRoom)
   - `room.players[]` — player info with presenceState

4. **Player hand display:**
   - Show viewer's hand cards as clickable card elements
   - Cards show rank and suit with color (red for hearts/diamonds, black for spades/clubs)
   - Selected card is highlighted

5. **Table cards display:**
   - Show face-up table cards in a grid
   - Last discarded card highlighted or separated

6. **Action buttons:**
   - "抽牌" (Draw) — emit `pickred:draw`
   - "撿牌" (Pick Discard) — emit `pickred:pick-discard` (only if lastDiscard exists)
   - "配對" (Match) — emit `pickred:match` with handCardId and tableCardId (only when a valid match is selected)
   - "出牌" (Discard) — emit `pickred:discard` with cardId (only when a hand card is selected)

7. **Turn indicator:**
   - Show whose turn it is with a countdown timer
   - Highlight current player's area
   - Show "你的回合" / "對方回合" text

8. **Scoring display:**
   - Show both players' collected red-card scores
   - Show collected card counts

9. **Ready/Waiting state:**
   - When `stage === "waiting"`: show "準備" button that emits `pickred:ready`
   - Show player list with ready status

10. **Finished state:**
    - When `stage === "finished"`: show winner, final scores, "再來一局" button

11. **Reconnection:**
    - Use `data-presence-state` attribute for presence styling
    - Show reconnection banner when presence is "reconnecting"
    - Re-subscribe on socket reconnect

12. **Create `styles/PickRedRoom.module.css`:**
    - Card styling: rank/suit display, red/black colors, selected state
    - Table layout: table cards grid, hand cards row
    - Turn indicator: active player highlight, countdown
    - Score display: side-by-side scores
    - Responsive layout for mobile</action>
