       - Create the lap trigger body: CANNON.Body with mass 0, isTrigger=true, CANNON.Box shape (TRACK_WIDTH/2, 5, 0.1), positioned at trackDef.lapTriggerPosition

       - `input` = { accel: 0|1, brake: 0|1, steer: -1..1 }

       - If accel: apply force = forward.scale(ACCEL_FORCE) at carBody.position
       - If brake: apply force = forward.scale(-BRAKE_FORCE) at carBody.position
       - If steer != 0: create rotation quaternion from euler (0, STEER_RATE * steer * dt, 0), multiply with carBody.quaternion
       - Speed cap: if velocity.length() > MAX_SPEED, scale velocity to MAX_SPEED
       - Lateral friction: compute right vector, project velocity onto it, apply impulse to cancel lateral component * FRICTION_FACTOR
       - Constants: ACCEL_FORCE=800, BRAKE_FORCE=600, STEER_RATE=2.5, MAX_SPEED=30, FRICTION_FACTOR=0.85

       - Check direction: compute forward from car quaternion, reject if forward.z > 0 (wrong direction on the straight)
       - Increment lap count for that car
       - Return cleanup function to remove listener
