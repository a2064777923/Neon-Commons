<threat_model>
- **T-06-03:** Playwright passes against an isolated debug port while the deployed `3100` container remains stale, giving a false release signal.
  **Mitigation:** Make `3100` redeploy the default behavior of the release runner and reserve alternate ports for debugging only.
- **T-06-04:** Browser smoke starts before the frontend/backend pair is ready, causing avoidable flake.
  **Mitigation:** Add explicit readiness checks for the deployed stack before running browser suites.
- **T-06-05:** Operators keep stitching together ad hoc release commands even after the suite scripts exist.
  **Mitigation:** Add a single top-level `verify:release` entry point and make later docs point only at that command.
</threat_model>
