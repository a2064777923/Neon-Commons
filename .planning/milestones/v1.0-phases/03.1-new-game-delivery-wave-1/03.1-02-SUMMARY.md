## Decisions Made

- Sokoban intentionally bypasses the multiplayer lobby shell and room-entry semantics to keep direct-launch behavior honest.
- The first bundled level is solvable in one move so smoke coverage can prove end-to-end direct-launch behavior quickly and deterministically.
- Solo UI state stays thin because the rules module owns counters and transition semantics.
