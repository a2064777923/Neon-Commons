# 產品需求

## 玩家側

- 註冊 / 登入 / 退出
- 查看個人虛擬豆、段位分、勝負場
- 從遊戲庫選擇不同遊戲入口
- 創建房間並選擇規則模板
- 通過六位房間號加入房間
- 房主可補機器人進場
- 對局支持叫分、出牌、過牌、托管、自動結算
- 支持快捷文字消息與表情
- 支持人工操作限時與托管隨機出牌延時分離配置
- 支持房內語音通話
- 狼人殺支持角色分配、補 AI、晝夜節奏、獵人翻槍與投票出局
- 阿瓦隆支持補 AI、組隊、表決、任務投票與刺殺階段
- 五子棋支持 15 路棋盤、房號進房、補 AI 與實時落子
- 跳棋支持 2 人中國跳棋星盤、連跳合法落點高亮與補 AI

## 排行榜

- 按段位分排序
- 次序使用虛擬豆、勝場作為次級排序

## 後台

- 查詢玩家
- 調整虛擬豆、段位分
- 封禁 / 解封
- 修改模板是否上線
- 修改模板 settings JSON
- 修改系統配置
- 可直接調整底分、炸彈倍率、托管相關時間配置

## 首版遊戲範圍

- 斗地主：完整主玩法與排行榜
- 狼人殺：角色池 `狼人 / 預言家 / 女巫 / 守衛 / 獵人 / 村民`
- 阿瓦隆：角色池 `梅林 / 派西維爾 / 忠臣 / 刺客 / 爪牙 / 莫甘娜 / 莫德雷德 / 奧伯倫`
- 五子棋：`15 路棋盤 / 五連即勝 / 2 人對局`
- 跳棋：`2 人中國跳棋 / 星盤走位 / 進營獲勝`
