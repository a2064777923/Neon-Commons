# 五子棋 / 跳棋規則說明

## 在線五子棋

- 支持 `2` 人
- 棋盤為 `15 x 15`
- 黑棋先手，白棋後手
- 任一方先形成連續 `5` 子即獲勝
- 棋盤落滿且無五連時判定和局
- 房主可在等待階段補 AI 入場，雙方準備後立即開局

## 在線跳棋

- 當前為 `2` 人中國跳棋版
- 使用標準星盤，雙方分別佔據上下對角營地
- 單回合可以：
  - 走一步到相鄰空位
  - 經由一個或多個棋子連跳至空位
- 勝負：
  - 先把自己的全部棋子送入對側營地者獲勝
- 房主可在等待階段補 AI 入場，雙方準備後立即開局

## 對局節奏

- 兩種棋類房都帶單回合倒計時
- 倒計時結束時，系統會按當前局面自動代走一步，避免房間卡死
