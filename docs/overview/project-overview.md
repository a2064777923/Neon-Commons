# 項目概覽

本項目是單機容器化部署的派對遊戲平台，採用 `Next.js 前端 + Node.js / Socket.IO 後端 + PostgreSQL`。

核心能力：

- 玩家帳號註冊登入
- 遊戲庫首頁與多遊戲入口
- 房號加入、公開房列表與統一房間頁框架
- 三人斗地主實時對局
- 在線狼人殺與在線阿瓦隆實時房間
- 在線五子棋與在線跳棋實時房間
- 沉浸式遊戲房間頁，包含動畫牌桌 / 階段桌面、快捷消息與語音席
- 機器人補位與托管（斗地主）
- 狼人殺 / 阿瓦隆等待階段補 AI 與自動對局
- 五子棋 / 跳棋等待階段補 AI 與自動代走
- 排行榜
- 管理後台玩家與規則配置

首版限制：

- 支持 `CLASSIC`、`ROB`、`NO_SHUFFLE` 三種斗地主模板
- 支持狼人殺與阿瓦隆擴展角色與可玩流程
- 支持五子棋與 2 人中國跳棋可玩流程
- `LAIZI` 模板已預留配置，但默認未開放
- 前後端雖已分離，但仍在同一倉庫與單機部署模型中運行
- 房間狀態當前保存在單節點進程內存，適合本機部署
- 人工回合倒計時與托管自動出牌延時分離，便於做出更像實機的托管節奏
